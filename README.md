# 🏡 Casa Salento — Landing Page
---
https://casasalento.netlify.app/
---

Landing page moderna e responsive per una casa vacanze situata a Santa Maria al Bagno (LE), nel cuore del Salento.

Progetto sviluppato con focus su **UI/UX, modularità e performance**, pensato per presentare la struttura in modo chiaro, elegante e convertire visite in contatti diretti (WhatsApp).

---

## ✨ Features

- 🌐 **Multilingua (IT / EN)** con sistema i18n custom
- 📱 **Responsive design** (mobile-first)
- ⚡ **Performance ottimizzate** (CSS minificato, JS modulare)
- 🧩 **Architettura modulare** (config, translations, main)
- 💬 **Contatto diretto WhatsApp**
- 🖼️ **Gallery interattiva + lightbox**
- 🎯 **SEO base + Open Graph + Schema.org**
- 🎨 **UI moderna** con icone Lucide

---

## ⚙️ Configurazione

Tutti i link dinamici sono centralizzati in /js/config.js

👉 Modificando questo file puoi aggiornare:

- numero WhatsApp
- link Facebook
- Google Maps
- messaggi precompilati

senza toccare HTML o JS principale.

---

🌍 Traduzioni

Sistema i18n custom basato su attributi data-i18n:

File:

- /js/translations/it.js
- /js/translations/en.js

Cambio lingua:

- salvato in localStorage
- auto-detect browser
